# Store Assets Bundle

This folder tracks required metadata and art deliverables for App Store and Play Store submissions.

## Required Outputs
- App icons (all required sizes)
- Launch/splash screens
- App Store screenshots (6.7", 6.5", 5.5", iPad)
- Play Store screenshots (phone/tablet)
- Feature graphic (1024x500)
- Promotional video link (optional)

Use `scripts/generate-store-assets.sh` as a starter helper.
