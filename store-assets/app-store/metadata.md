## Description
AI Assassins is a tactical daily briefing app for operators, founders, and high-accountability teams.

Get one command-ready dashboard with:
- Overnight overview and priority narratives
- Markets snapshot (S&P 500, NASDAQ, WTI, BTC)
- Local weather context
- Calendar next-up items from ICS
- Scripture and reflection
- Mission priorities, top tasks, and command note

Subscription tiers:
- Free: limited daily brief generation
- Premium: expanded generation limits, history, and export workflows

Important:
- AI output is informational and should be independently verified.
- Not financial, legal, medical, or emergency advice.
