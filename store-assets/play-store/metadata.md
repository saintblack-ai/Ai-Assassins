## Short Description

## Full Description
AI Assassins delivers a structured command briefing each day.

Core sections include:
- Overnight Overview
- Markets Snapshot (S&P 500, NASDAQ, WTI, BTC)
- Local Weather Summary
- Next Up Calendar (ICS)
- Scripture of the Day
- Mission Priorities
- Truthwave narrative/risk/counter
- Top Tasks + Command Note

Monetization:
- Free tier with limited usage
- Premium tier for expanded access and advanced workflows

Compliance and trust:
- Authentication-gated generation
- Server-side quota and entitlement checks
- No API keys in client builds

AI-generated content may contain errors and should be reviewed before operational use.
